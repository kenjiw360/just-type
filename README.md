# Just Type!
*A Note Taking AI that categorises your notes as you type*

Technologies Used:
- Vector Database (PostgreSQL, self hosted)
- Text Embedder (ANGLE)
- ReactJS (for note taking app)

After downloading dependencies, run with `npm start`!
